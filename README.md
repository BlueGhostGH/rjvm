# RJVM

JVM in Rust that can only parse ["Hello, World"](https://en.wikipedia.org/wiki/%22Hello,_World!%22_program) inspired by [tsoding's JelloVM](https://github.com/tsoding/JelloVM).

## Quick Start

```console
$ javac Main.java
$ cargo run
```
